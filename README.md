# Padawan

Configure JEDI to take TOX virtual env account

# Dependencies

* projectile 0.13.0
* Jedi 0.2.7

# Installation

Download `padawan.el` and ensure it is in a directory in your `load-path`.

Put something like this in your .emacs

	(require 'padawan)

# Running

Triggered during `projectile-switch-project`
